# Components Folder Structure

Quick reference for navigating the portfolio codebase.

## Layout (`components/layout/`)

Navigation, headers, footers—fixed or repeated across the page.

| File          | Description                    |
|---------------|--------------------------------|
| TopMenuBar    | Top navigation bar             |
| SidebarNav    | Desktop sidebar navigation     |
| NavXian       | Mobile bottom navigation       |
| Navigation    | Alternate nav component        |
| Footer        | Main footer                    |
| FooterXian    | Mobile footer variant          |

## Sections (`components/sections/`)

Main page sections—each maps to a visible block on the homepage.

| File              | Description                    |
|-------------------|--------------------------------|
| Hero              | Hero / intro section           |
| About             | About Me                       |
| Experience        | Work experience                |
| KeyDifferentiators| Signature strengths            |
| Projects          | Projects showcase              |
| Skills            | Technical skills               |
| Education         | Education & coursework         |
| Certifications    | Certifications                 |
| Contact           | Contact form & info            |

## Mobile (`components/mobile/`)

Mobile-specific or alternate layout components.

| File          | Description                    |
|---------------|--------------------------------|
| Home          | Alternate home layout          |
| ProjectsXian  | Mobile projects variant        |
| ContactXian   | Mobile contact variant         |
| ServicesXian  | Mobile services variant        |
| ExperienceXian| Mobile experience variant      |

## UI (`components/ui/`)

Reusable UI pieces and utilities.

| File        | Description                    |
|-------------|--------------------------------|
| CountUp     | Number count-up animation      |
| ScrollUp    | Scroll-to-top button           |
| LoadingScreen | Initial loading screen       |
